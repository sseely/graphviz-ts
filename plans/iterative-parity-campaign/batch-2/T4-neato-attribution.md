## Context

`test/corpus/parity-neato.json` (2026-07-11 sweep): 762 total, 263
pass, 492 diverged, 7 oracle-error, 0 port-error, 0 timeout, tolerance
0.5. This task runs batch-1's injection harness across the full

## Task

1. Run `test/corpus/attribute-divergence.ts neato` (T1, batch-1) to
   completion — full diverged set, no `--fresh` needed for a first run.
   Verify it resumes cleanly if interrupted.
2. Run `test/corpus/oracle-error-classifier.ts neato` (T3, batch-1) on
   the 7 `oracle-error` rows.
3. Write `batch-2/neato-buckets.md`: rank `bucket.shape` values by
   member count (largest first), and for each bucket give: member
   count, one representative id, and the `firstDiff` string for that
   representative. Separately call out any bucket where
   `bucket.uniformDelta` or `bucket.mirror` fired for most/all members
   — those are strong `A1-drift` candidates per D5.
4. Do not fix anything in this task — attribution and bucketing only.
   Fixes happen in batch-3.
