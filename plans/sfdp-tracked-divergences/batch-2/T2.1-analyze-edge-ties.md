# T2.1 — Analyze the B2 edge FP-ties

## Context
Edge spline divergences in sfdp route through either `makeMultiSpline` (GTS
CDT triangulation, `src/layout/neato/multispline.ts` + `src/*` CDT) or the
flat/straight edge path. The known failure signature is an FP tie (incircle
predicate, ccw orientation, hypot tie-break) or a segment-count `round()`
straddle — both flip a discrete routing decision.

## Task
For each B2 representative (42, 241_0, 2521_1) and each other distinct B2 input:
inject exact native positions, then instrument the routing predicate to dump
the deciding value on both sides (per `diagnosis.md`: instrument before
hypothesizing). Identify the exact predicate/site and whether the port's value
differs from the C binary's at the ULP that flips the decision. Record
`fixable:<site + expected contraction>` or `irreducible:<disasm evidence>`.

## Read-set
- `src/layout/neato/multispline.ts`, `src/layout/neato/splines.ts`
  (RoutingHelper), the CDT incircle/ccw predicate sources
- `src/common/fma.ts`, `src/common/arm-pow.ts` (contraction emulation)
- flat path: `src/layout/dot/straight-edges.ts`, make_flat_edge sites
- memory: [[multispline-port-landed]], [[fma-ccw-emulated]],
  [[flat-edge-241-is-y-only]], [[2368-flat-geom-getmainedge-eddist-done]]

## Write-set
- `batch-2/findings.md`

## Method
1. Inject exact positions; confirm the edge still diverges (isolates routing
   from drift).
2. Instrument the deciding predicate (incircle / ccw / segment-count) to print
   the port value; compare to a C dump (instrument the native predicate, or use
   `otool -tvV` on the plugin per [[fma-ccw-emulated]]).
3. State the mechanism: which predicate, which `file:line`, why the port's
   result differs (missing fma? round() mode? predicate non-robustness?).

## Acceptance criteria
- Given each B2 input, when instrumented, then findings.md names the deciding
  predicate + `file:line` + the port-vs-C deciding value.
- Given a fixable verdict, when written, then it specifies the exact
  contraction/predicate change T2.2 will make.
- Given an irreducible verdict, when written, then it cites disassembly (fmadd)
  or a native-vs-V8 libm ULP probe — not an assertion.
